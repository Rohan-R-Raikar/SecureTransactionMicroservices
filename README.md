# Secure Transaction Microservices Platform

## Overview
This project demonstrates a secure microservices-based platform built using ASP.NET Core.
It focuses on authentication, service-to-service security, background processing,
and real-time notifications.

## Architecture
- Microservices-based architecture
- API Gateway using Ocelot
- Database-per-service pattern
- Secure communication using JWT and HMAC

## Services
- Auth Service
- Order Service
- Notification Service

## Security
- JWT authentication
- HMAC validation at API Gateway
- AES encryption for sensitive data
- Rate limiting

## Tech Stack
- ASP.NET Core
- Ocelot API Gateway
- Hangfire
- SignalR
- EF Core
- SQL Server
